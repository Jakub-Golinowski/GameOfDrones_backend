Game Of Drones backend homepage.
List of endpoints:
  • /map_2d - JSON data for all aircrafts (including drones) but each entry is pruned to contain only latitude, longitude, altitude and aircraft icao number.
  • /mirror_recorded - simply mirrors the involi feed (https://hackzurich.involi.live/http_recorded/).
  • /post_drone_position - URL for POST or PUT requests with updated drone data. The form of the request should be the following JSON format: { "icao": "FA4548", "lat": 46.530834474, "lon": 6.2578792135, "alt": 9753, "speed": 236.129796, "heading": -51.76, "last_update": "2018-09-14T22:42:25.898475+00:00" }
  • /post_drone_delete - URL for drone deletion by passing its 24 bit icao number (in HEX). For example with such a command: curl --header "Content-Type: application/json" --request POST --data 'FA4548' http://0.0.0.0:3000/post_drone_delete
  • /full_data - URL for full air traffic data (both from drones and from the planes)
